# 🚀 TechLearn QOTD (Question of the Day Platform)

## 📌 Overview
TechLearn QOTD is a full-stack web application that provides a "Question of the Day" system for users to practice coding problems, submit solutions, and improve their skills consistently.

---

## 🧠 Features
- 📅 Daily coding questions (QOTD)
- ✍️ Code submission system
- 🧪 Run & evaluate solutions
- 🔐 User authentication
- 📊 Track progress
- 🎨 Smooth UI with animations

---

## 🏗️ Tech Stack

### 💻 Frontend
- React (Vite)
- Tailwind CSS
- Framer Motion
- Lucide Icons

### ⚙️ Backend
- Node.js
- Express.js

### 🗄️ Database (if used)
- MongoDB / JSON-based storage
